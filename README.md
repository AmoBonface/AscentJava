## 💡 How It Works

1. The program asks for your **name**
2. The program asks for your **age**
3. You enter your **weight**
4. You enter your **height**
5. You choose your **unit** (metric or imperial)
6. You enter your **gender**
7. It calculates your BMI and returns:

   > Name, Age, Gender, Unit, Weight, Height, BMI, Category, Advice, and Motivation.

---

## ▶️ How to Run

**Requirements:** Java must be installed on your machine.

```bash
# Step 1: Clone the project
git clone https://github.com/AmoBonface/AscentJava.git

# Step 2: Navigate into the folder
cd mambo-bmi-calculator

# Step 3: Compile the program
javac Mambo.java

# Step 4: Run it
java Mambo
```
